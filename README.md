# dentali-rest
[![Licencia](https://img.shields.io/badge/Licencia-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Estado](https://img.shields.io/badge/Estado-En%20Desarrollo-brightgreen)]()

Dentali-rest es un proyecto que pretende generar una implementación sobre métodos generalizados de un 
sistema de gestión de consultorios medicos y/o dentales para facilitar el manejo de las citas e historias
clínicas de los pacientes.

El presente proyecto ha sido creado sin fines de lucro.
